
## Doel

Alias in bash instellen. Het command dat wordt aangeroepen is het shell script.

---
# Alias configuratie (.bashrc)

Om het script snel te starten, wordt een alias gebruikt.

## Stap 1 — Open `.bashrc`

In WSL:

```bash
nano ~/.bashrc
```

---

## Stap 2 — Voeg alias toe

Voorbeeld:

```bash
alias demo11="bash /mnt/c/Users/abc/Les-week-11-Kubernetes-Orchestratie/demo-week-11.sh"
```

---

## Stap 3 — Herlaad configuratie

```bash
source ~/.bashrc
```

---

## Stap 4 — Gebruik

```bash
demo11
```

👉 start direct de demo
