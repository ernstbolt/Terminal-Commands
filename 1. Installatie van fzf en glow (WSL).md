
## Doel

Deze tools worden gebruikt voor een interactieve CLI demo-ervaring:

- **fzf** → selecteren van commands (menu)
    
- **glow** → tonen van markdown (preview)
    

Samen vormen ze een “terminal notebook”, ideaal voor bijvoorbeeld kubectl demo’s.

---

## Overzicht tools

- fzf  
    Interactieve fuzzy finder voor selectie in de terminal
    
- glow  
    Renderen van markdown in de terminal (voor preview)
    

---

# Installatie in WSL

## Stap 1 — Update package lists

```bash
sudo apt update
```

---

## Stap 2 — Installeer fzf

```bash
sudo apt install fzf -y
```

Controleer installatie:

```bash
fzf --version
```

---

## Stap 3 — Installeer glow (via snap)

Installeer glow met snap:

```bash
sudo snap install glow
```

Controleer installatie:

```bash
glow --version
```
