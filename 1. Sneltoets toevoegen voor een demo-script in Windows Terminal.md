## Doel

Met een sneltoets in Windows Terminal kun je een PowerShell-commando direct laten invoeren, bijvoorbeeld `demo11`, waarna je WSL-script start. Windows Terminal ondersteunt hiervoor keybindings in de instellingen, en `sendInput` is daar de juiste actie voor. Een keybinding moet bestaan uit modifiers plus een echte toets; alleen `Ctrl+Shift` is dus niet geldig. ([Microsoft Learn](https://learn.microsoft.com/en-us/windows/terminal/customize-settings/actions?utm_source=chatgpt.com "Custom actions and keybindings in Windows Terminal"))

## Stap 1 — Open Windows Terminal instellingen

Open Windows Terminal en druk op:

```text
Ctrl + ,
```

## Stap 2 — Open `settings.json`

Open linksonder **Open JSON file**, of gebruik de Shift-methode hierboven. Het `settings.json`-bestand staat normaal in het LocalState-pad van Windows Terminal. ([Microsoft Learn](https://learn.microsoft.com/en-us/windows/terminal/install?utm_source=chatgpt.com "Install and get started setting up Windows Terminal"))

Gebruikelijk pad:

```text
%LOCALAPPDATA%\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json
```

## Stap 3 — Voeg een actie toe

Voeg in de `actions`-array deze binding toe:

```json
{
  "command": {
    "action": "sendInput",
    "input": "demo11\r"
  },
  "keys": "ctrl+alt+1"
}
```

`sendInput` is bedoeld om tekst naar je shell te sturen via een keybinding. In dit geval stuurt het `demo11` plus Enter (`\r`) naar PowerShell. Microsoft laat dit patroon expliciet zien in de documentatie. ([Microsoft Learn](https://learn.microsoft.com/en-us/windows/terminal/tips-and-tricks?utm_source=chatgpt.com "Windows Terminal tips and tricks"))

## Voorbeeld van een compleet `actions`-blok

```json
"actions": [
  {
    "command": {
      "action": "sendInput",
      "input": "demo11\r"
    },
    "keys": "ctrl+alt+1"
  }
]
```
